Project note Pi-hole on Xfinity

Goal

Use a Raspberry Pi as network-wide DNS filtering for home devices.

What broke

Xfinity kept devices on Comcast DNS, and IPv6 could bypass the setup entirely.

How I debugged it

Tested Pi-hole directly with nslookup, checked dashboard queries, then isolated router limits.

What it proves

I understand request flow, DHCP control, IPv4/IPv6 behavior, and safe testing before global changes.
